# Aurelia Hotel

## Opis projekta

Web sajt Aurelia Hotel predstavlja responzivni hotel sajt sa informacijama, kontakt formom i sistemom rezervacija.

## Tehnologije

HTML, CSS, JavaScript

## Funkcionalnosti

- Promena teme (crna/svetla/zlatna)
- Dinamičko dodavanje ponuda
- Validacija kontakt forme
- Sistem rezervacija

## Stranice

- Početna
- O nama
- Kontakt
- Rezervacije

## Pokretanje projekta

Otvoriti index.html u pretraživaču.
